Calculator Web Application

Description

A simple and responsive Calculator Web Application developed using HTML, CSS, and JavaScript. This calculator performs basic arithmetic operations such as addition, subtraction, multiplication, and division through an interactive user interface.

Features

- Addition, subtraction, multiplication, and division
- User-friendly interface
- Responsive design
- Clear (AC) function
- Delete (DE) function
- Real-time calculation display

Technologies Used

- HTML
- CSS
- JavaScript

How to Use

1. Open the calculator in a web browser.
2. Click the number buttons to enter values.
3. Select an arithmetic operator.
4. Press "=" to display the result.
5. Use "AC" to clear all inputs.
6. Use "DE" to delete the last entered digit.

Project Objective

The objective of this project is to understand and implement JavaScript-based arithmetic operations, event handling, and responsive web design concepts.
